# Product Images Folder

Place your product images in this folder.

## How to Use:

1. **Add your product images** to this folder (`public/products/`)
   - Example: `public/products/alsa-for-men.jpg`
   - Example: `public/products/alsa-velmir.jpg`

2. **In the Admin Panel**, use the image path:
   - Main Image: `/products/alsa-for-men.jpg`
   - Additional Images: `/products/img1.jpg, /products/img2.jpg`

## Image Naming Tips:

- Use descriptive names: `alsa-for-men.jpg`, `alsa-velmir.jpg`
- Use lowercase and hyphens: `product-name.jpg`
- Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`, `.gif`

## Example URLs:

- `/products/alsa-for-men.jpg`
- `/products/alsa-velmir.png`
- `/products/alsa-midnight.webp`

## Notes:

- Images in this folder are publicly accessible
- After adding images, commit them to git and push
- Images will be available at: `https://your-domain.com/products/image-name.jpg`
