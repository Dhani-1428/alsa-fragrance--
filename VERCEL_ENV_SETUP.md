# Vercel Environment Variables Setup

## Cloudinary Credentials for Vercel

Add these environment variables in your Vercel project:

1. Go to: **Vercel Dashboard** → **Your Project** → **Settings** → **Environment Variables**

2. Add these 3 variables:

```
CLOUDINARY_CLOUD_NAME = dhsxlkq79
CLOUDINARY_API_KEY = 784935484669973
CLOUDINARY_API_SECRET = uulvT-66pvsQ2iDrKuriWrF1rYU
```

3. Select all environments: **Production**, **Preview**, and **Development**

4. Click **Save**

5. **Redeploy** your application for changes to take effect
